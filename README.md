## ğŸš¦ Getting Started

1. **Clone the repository**:

    ```bash
    git clone https://github.com/argha-paul-2002/MERN_Doctor_Booking_Backend.git
    ```

2. **Install dependencies**:

    ```bash
    cd medico-backend
    npm install
    ```

3. **Configure environment variables**:

